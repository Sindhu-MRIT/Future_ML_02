Future_ML_02
Customer Support Ticket Classification System

Built an ML system that reads support tickets and automatically figures out what category they fall under and how urgent they are. Trained on 47,000 real IT support tickets. The model classifies tickets into 8 categories and tags each one as High, Medium or Low priority.

Tools used: Python, Pandas, numpy, re, streamlit, NLTK, Scikit-learn, Matplotlib, Seaborn, train_test_split, confusion_matrix, accuracy_score

Main Attachments:
- ticket_classification.ipynb - full code and results
- category_distribution.png - breakdown of ticket types
- priority_distribution.png - priority level breakdown
- confusion_matrix.png - model performance visualization
- visual2_app.py - full code for visual interface
- Dashboard URL Access - https://futureml02-bdd5h6p2z5osec3xxgxqkm.streamlit.app/

Dataset: IT Service Ticket Classification Dataset from Kaggle
